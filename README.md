# 🐉 Dragon Slayer  
**Dragon Slayer** is a 2D action platformer built with [Godot Engine](https://godotengine.org). Players take on the role of a valiant hero tasked with slaying dragons, overcoming challenging obstacles, and collecting treasures in a pixel-art fantasy world.

---

## 🎮 Features  
- **Exciting Gameplay**: Engage in intense dragon battles and platforming challenges.  
- **Pixel-Art Graphics**: Stunning handcrafted pixel visuals inspired by retro classics.  
- **Dynamic Levels**: Navigate through multiple levels filled with enemies, traps, and secrets.  
- **Collectibles**: Gather coins, treasures, and power-ups to enhance your abilities.  
- **Epic Soundtrack**: Immerse yourself in a rich and atmospheric music score.
